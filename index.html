How It Works

The Fake News Detector uses machine learning and external verification to identify potentially fake news articles.

Detection Process

  1. Text Analysis: The system processes the news text using natural language processing techniques
  2. Machine Learning: A custom logistic regression model analyzes the text features
  3. External Verification: Optionally checks against NewsAPI for corroborating sources
  4. Result Synthesis: Combines all analysis to provide a authenticity verdict

Technical Features

  • Custom TF-IDF implementation with n-grams
  • Synonym mapping and stemming
  • Dynamic stopword filtering
  • Cosine similarity for external article matching
  • Semantic similarity using Sentence Transformers
